# 🤖 AI Multi-Agent Personal Assistant

🚀 A powerful AI-driven system that uses multiple intelligent agents to assist with productivity, coding, resumes, interviews, and career growth.

---

## 🌟 Key Features

- 🤖 Multi-Agent Architecture (Planner, Coding, Resume, Interview, Productivity)
- 📄 Resume Analysis & ATS Score Calculator
- 💬 Chat History Tracking
- 🔐 Authentication System
- 📊 Analytics Dashboard
- 📁 PDF Parsing for Resume Processing
- 🧠 Smart Career Guidance System

## 🧠 How It Works

1. User logs in  
2. Provides a goal  
3. AI agents collaborate:
   - Planner Agent → roadmap  
   - Coding Agent → coding help  
   - Resume Agent → resume improvement  
   - Interview Agent → interview prep  
   - Productivity Agent → tracking

## 🚀 Installation & Setup

```bash
git clone https://github.com/vaseema-shaik/AI-Multi-Agent-Personal-Assistant.git
cd AI-Multi-Agent-Personal-Assistant

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt
python app.py
```
